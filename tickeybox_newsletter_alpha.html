Feature 01
You control
every number

Your child's phone can only call — and be called by — people you've approved. Nobody gets through unless you've added them first.

For each contact you set two things: can they call your child, and can your child call them back? Both default to yes. You can restrict either.

Feature 02
You set
when it rings

Quiet Hours lets you define a window when the phone simply doesn't ring. No calls in, no calls out. You set it once; it runs every night.

Set a from/to time. Choose a message for anyone who calls during that window. Emergency contacts always get through — regardless of the time.

Feature 04
One account,
every phone

If you have more than one TickeyBox in the house — the kitchen, a bedroom — they're all managed from a single login. See which ones are online at a glance.

Each phone has its own contacts, its own quiet hours, its own call history. Manage them independently or copy settings across.

Multiple phones One login Separate settings per phone

A few things worth being honest about: the labels in the app aren't parent-friendly yet. Some still say things like "Inbound / Outbound" and "Offline Message" — written by engineers, for engineers. We've logged every one of them and they're being fixed before the pilot goes live.

What does work: every feature you see above is live and functional. We've been calling it from real phones. The flows complete. The contacts block and allow correctly. The quiet hours cut at the right time. We're obsessing over the function first — the polish is coming.
